# Ivykids-Task
Full Stack Application -> Twitter-Clone

## Technologies Used : MERN stack application

- React JS
- React Context API + useReducer
- React Router v6
- Tailwind CSS
- NodeJS and ExpressJS
- Material UI components
- MongoDB with Mirage for backend
- React Icons
- React Hot Toast
